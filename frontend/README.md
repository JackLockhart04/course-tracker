## My project

I wanted to create a website to track my courses easier by showing grade and letter grade with custom ABC scales. I also wanted more functionality like checking what grade I need on remaining assignments to get a certain grade or letter grade. Finally I want to be able to track my total gpa and grades needed in remaining courses to get a certain gpa

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
